   - Hybrid 的目标是"全能"，不是"单项极致"
   - 在直线上牺牲了一些精度
   - 换取了曲线上的最优表现
   - 这是必要的设计权衡

【为什么 Line 这么差？（写报告时的解释）】

  - 每次实验结果完全不同
  - 系统行为不可预测
  - 可重复性极差（24.5mm）

结论：Line 完全不适合直线跟随

【常见问题解答（给组员）】

Q1: 为什么 Bump 在直线上这么优秀？
A: 因为 Bump 的线性测距能力强，距离变化直接映射到速度控制，提供了
   极高的控制精度。在直线任务中，只需要保持距离即可，Bump 的控制
   策略完美适配。

Q2: 为什么 Hybrid 在直线上不如 Bump？
A: 因为 Hybrid 包含了 Line 部分，而 Line 在直线上信号不稳定，引入了
   额外的噪声，导致航向波动增大和轨迹不够直。但 Hybrid 仍然远超 Line。

Q3: 为什么 Hybrid 的路径直线度反而最差？
A: 因为 Line 部分在直线上会提供不稳定的转向信号，这些信号反而破坏了
   轨迹的直线性。这是为曲线性能付出的代价。

Q4: 为什么 Line 在直线上这么差？
A: 因为 Line 不擅长测量距离，只能检测横向位置差异。在直线上，横向
   位置差异很小，信号弱且噪声大，导致控制不稳定。

Q5: 假设 C 在直线上验证了什么？

   1) Bump 在直线上最优（所有 4 个指标都最好）

   3) Line 在直线上极差（完全不适合）
   4) 结合曲线数据，Hybrid 是全能冠军
