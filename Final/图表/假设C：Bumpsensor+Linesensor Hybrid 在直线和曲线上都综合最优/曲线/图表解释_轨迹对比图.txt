图表解释：三传感器 30°曲线轨迹对比图
================================================================================

【这张图在做什么？】
这张图对比了三种传感器在走 30° 曲线时的表现：
- 左边：Line Sensor（绿色）- 曲线专家，表现很好
- 中间：Bump Sensor（红色）- 曲线苦手，表现很差
- 右边：Bump+Line Hybrid（蓝色）- 全能选手，表现最稳定！

【实验设置】
- 任务：跟随机器人走 30° 曲线路径
- 每个传感器：10 次重复实验
- 目标：找出哪个传感器在曲线上最稳定、最可靠

================================================================================
【上半部分：平均轨迹 + 95% 置信区间（CI）】
================================================================================

这是什么？
  - 粗线：10 次实验的平均轨迹
  - 阴影区：95% 置信区间（数据分布范围）

左图（绿色 - Line Sensor）：优秀

  - Y 轴范围：-180mm 到 +150mm（共 330mm）

右图（蓝色 - Bump+Line Hybrid）：最优秀！

  - 从 Y=0 平滑下降到 Y=-90mm

优点 2：置信区间最窄！

优点 3：轨迹最稳定
  - 平均轨迹非常平滑
  - 没有突变，没有波动

结论（上半部分）：

【为什么 Hybrid 最好？】
================================================================================

Hybrid 的优势（曲线跟随）：

1. 结合了 Line 和 Bump 的优点
   - Bump 部分：提供稳定的距离控制
     → 保持合适的跟随距离，不会过近或过远
   
   - Line 部分：提供精确的方向控制
     → 检测横向位置偏移，精确调整转向
   
   - 时分复用架构：两者配合无干扰
     → Bump 窗口（100ms）+ Line 窗口（100ms）交替运行

2. 弥补了单一传感器的缺陷
   - Line 单独使用时：距离估计不准确
     → Hybrid 用 Bump 提供准确距离
   
   - Bump 单独使用时：方向判断混乱
     → Hybrid 用 Line 提供准确方向

3. 实现了最优的控制策略
   - 速度控制：由 Bump 提供（线性测距优势）
   - 转向控制：由 Line 提供（横向检测优势）
   - 两者独立控制，不互相干扰
   - 达到最优的速度-转向平衡

4. 数据证明
   - 置信区间最窄（±5mm）
     → 一致性最好，最可靠
   
   - 轨迹最聚集
     → 可重复性最强
   
   - 曲线最平滑
     → 控制质量最高

【三者对比总结】
================================================================================

轨迹质量（曲线平滑度）：
  Hybrid ≥ Line >> Bump
  - Hybrid：完美的平滑曲线
  - Line：平滑曲线，略有波动
  - Bump：混乱无序，无法形成曲线

数据一致性（置信区间宽度）：
  Hybrid（±5mm）< Line（±8mm）<< Bump（±80mm）
  - Hybrid 最窄：最可靠
  - Line 较窄：较可靠
  - Bump 最宽：不可靠

可重复性（轨迹聚集度）：
  Hybrid > Line >> Bump
  - Hybrid：10 条线几乎重叠
  - Line：10 条线紧密聚集
  - Bump：10 条线到处乱飘

横向偏差范围：
  Line（75mm）< Hybrid（120mm）< Bump（330mm）
  - Line 最小（但可能走得不够远）
  - Hybrid 适中（走得更远，但控制稳定）
  - Bump 最大（完全失控）

综合评价：
  Hybrid：全方位最优，特别是稳定性和一致性
  Line：曲线质量好，但稳定性不如 Hybrid
  Bump：完全不适合曲线跟随

【特别注意：Hybrid 的"最窄置信区间"】
================================================================================

这是这张图最重要的信息！

置信区间（CI）的意义：
  - CI 窄 = 数据一致性好 = 可重复性强 = 系统可靠
  - CI 宽 = 数据分散 = 不可预测 = 系统不可靠

Hybrid 的 CI 最窄（±5mm）意味着：
  1. 每次实验结果高度一致
  2. 系统行为可预测
  3. 性能稳定可靠
  4. 适合实际应用

这是 Hybrid 最大的优势！
  - 不仅性能好，而且稳定可靠
  - 这是实际应用中最重要的指标
  - Line 虽然曲线好，但稳定性不如 Hybrid
  - Bump 更是完全不稳定
