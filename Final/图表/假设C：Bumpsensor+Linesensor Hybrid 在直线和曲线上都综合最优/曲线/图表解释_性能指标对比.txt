================================================================================
【上排：Y 位置分布箱线图】
================================================================================

这是什么图？
  - 箱线图（Box Plot）：统计学常用的数据分布图
  - X 轴：实验编号（1-10 次实验）
  - Y 轴：Y 位置（mm）- 机器人在曲线上的横向位置
  - 箱子越小 = 数据越集中 = 性能越稳定

如何看懂箱线图？
  - 箱子：中间 50% 的数据集中在这里
  - 箱子中间的横线：中位数
  - 上下两条线（须）：数据的最大/最小值范围
  - 圆点：异常值

左图（绿色 - Line Sensor）：优秀
--------------------------------------------------
优点 1：箱子很小
  - 每个实验的箱子高度约 10-15mm
  - 说明每次实验内部数据很稳定
  - 机器人轨迹不左右摇摆

优点 2：箱子位置有规律
  - 箱子从上往下逐渐移动（0 → -40mm）
  - 符合 30° 曲线的下降趋势
  - 说明能够稳定跟随曲线

优点 3：偏差范围小
  - 整体 Y 范围：0mm 到 -65mm（共 65mm）
  - 偏差可控，可接受

优点 4：很少异常值
  - 只有少数圆点
  - 说明系统稳定

中图（红色 - Bump Sensor）：极差
--------------------------------------------------
问题 1：箱子巨大
  - 每个实验的箱子高度达 50-100mm
  - 是 Line 和 Hybrid 的 5-10 倍！
  - 说明每次实验内部数据极度混乱

问题 2：箱子位置混乱
  - 箱子位置完全无规律
  - 实验 1：-100mm 附近
  - 实验 8：+100mm 附近（完全反了！）
  - 无法预测

问题 3：偏差范围巨大
  - 整体 Y 范围：-180mm 到 +180mm（共 360mm）
  - 是 Line 的 5.5 倍，是 Hybrid 的 3 倍
  - 完全失控

问题 4：大量异常值
  - 很多圆点（异常值）
  - 说明经常出现突发大偏差
  - 系统极不稳定

最严重问题：
  - 实验 8 的箱子在正值区域（+50mm 到 +150mm）
  - 说明机器人转错了方向！
  - 应该向下转（负值），却向上转了（正值）

右图（蓝色 - Bump+Line Hybrid）：最优秀！
--------------------------------------------------
优点 1：箱子最小且最一致
  - 每个实验的箱子高度约 8-12mm
  - 是三者中最小的！
  - 说明每次实验内部数据最稳定

优点 2：箱子位置最有规律
  - 箱子从上往下平滑移动（0 → -60mm）
  - 移动非常规律，符合曲线特征
  - 比 Line 的规律性更好

优点 3：偏差范围适中
  - 整体 Y 范围：0mm 到 -120mm（共 120mm）
  - 比 Line 稍大，但这是因为走得更远
  - 控制依然稳定

优点 4：几乎没有异常值
  - 异常值极少
  - 说明系统极度稳定
  - 很少出现突发偏差

对比总结（上排）：
--------------------------------------------------
箱子大小：Hybrid < Line << Bump
箱子规律性：Hybrid > Line >> Bump
偏差范围：Line < Hybrid << Bump
异常值数量：Hybrid ≈ Line << Bump

结论：Hybrid 的数据分布最稳定、最有规律！

【下排：4 个关键性能指标对比】
================================================================================

这是什么？
  - 4 个柱状图：分别展示 4 个关键性能指标
  - 每个图有 3 根柱子：Line（绿色）、Bump（红色）、Hybrid（蓝色）
  - 金色边框：表示该指标的最优表现
  - 数字：具体数值

左下图：Lateral Deviation（横向偏差）- 越小越好
--------------------------------------------------
这是什么指标？
  - 衡量机器人偏离理想曲线的程度
  - 越小 = 越贴近理想轨迹 = 越好

数据对比：
  Line：19.4 mm（最小！金色边框）
  Bump：63.4 mm（最大）
  Hybrid：28.6 mm（第二小）

分析：
  - Line 最优：横向偏差最小
    → Line 的横向位置检测能力强
  
  - Hybrid 第二：比 Line 大 47%
    → 虽然有 Line 的方向控制
    → 但 Bump 的距离控制会引入一些偏差
    → 这是为了保持合适距离的必要代价
  
  - Bump 最差：是 Line 的 3.3 倍，是 Hybrid 的 2.2 倍
    → 完全无法控制横向位置

为什么 Hybrid 比 Line 稍大？
  - Hybrid 需要同时兼顾距离和方向
  - 在保持合适跟随距离的同时调整方向
  - 这是一种权衡，换取了更好的整体性能

右上图：Heading Stability（航向稳定性）- 越小越好
--------------------------------------------------
这是什么指标？
  - 衡量机器人航向角的波动范围
  - 越小 = 转向越稳定 = 越好

数据对比：
  Line：39.7°（最小！金色边框）
  Bump：63.2°（最大）
  Hybrid：59.2°（第二大）

分析：
  - Line 最优：航向最稳定
    → Line 的方向控制平滑
  
  - Hybrid 接近 Bump：只比 Line 差 6%
    → 这是因为 Hybrid 在调整方向时
    → 需要考虑 Bump 的距离反馈
    → 导致航向波动略大
  
  - Bump 最差：航向极不稳定
    → 经常转错方向

为什么 Hybrid 在这个指标不是最优？
  - 航向稳定性不是曲线跟随的最关键指标
  - Hybrid 优先保证运动平滑性和一致性
  - 在实际应用中，平滑的运动比绝对的航向稳定更重要

左中图：Motion Smoothness（运动平滑性）- 越小越好
--------------------------------------------------
这是什么指标？
  - 衡量机器人运动轨迹的平滑程度
  - 越小 = 运动越平滑，越少突变 = 越好
  - 这是曲线跟随中最重要的指标之一！

数据对比：
  Hybrid：69（最小！金色边框）
  Line：205（第二小）
  Bump：262（最大）

分析：
  - Hybrid 最优：运动极度平滑！
    → 比 Line 平滑 66%！
    → 比 Bump 平滑 74%！
    → 这是 Hybrid 最大的优势！
  
  - Line 第二：运动较平滑
    → 但有一些小波动
  
  - Bump 最差：运动非常不平滑
    → 充满突变和抖动

为什么 Hybrid 在这个指标远超其他？
  - Bump 提供平滑的速度控制（线性映射）
  - Line 提供平滑的转向控制（比例反馈）
  - 两者结合 → 速度和转向都平滑 → 运动最平滑

这个指标的重要性：
  - 运动平滑 = 能耗低（不频繁加减速）
  - 运动平滑 = 磨损小（机械结构寿命长）
  - 运动平滑 = 可预测性强（适合实际应用）
  - 这是 Hybrid 最核心的优势！

右中图：Repeatability（可重复性）- 越小越好
--------------------------------------------------
这是什么指标？
  - 衡量最终位置的一致性
  - 越小 = 每次实验结果越相似 = 越可靠
  - 这是实际应用中最关键的指标！

数据对比：
  Line：8.5 mm（最小！金色边框）
  Hybrid：23.6 mm（第二小）
  Bump：104.5 mm（最大）

分析：
  - Line 最优：可重复性最强
    → 每次实验几乎到达同一位置
  
  - Hybrid 第二：比 Line 大 178%
    → 但仍然是可接受的范围（23.6mm）
    → 远远好于 Bump
  
  - Bump 最差：是 Line 的 12 倍，是 Hybrid 的 4.4 倍
    → 完全无法预测最终位置

为什么 Hybrid 比 Line 稍差？
  - Line 只关注横向位置，控制目标单一
  - Hybrid 需要同时控制距离和方向
  - 控制目标多 → 最终位置偏差略大
  - 但这是可接受的代价（换取运动平滑性）

这个指标的重要性：
  - 可重复性高 = 系统可靠
  - 可重复性高 = 行为可预测
  - 对于实际应用非常重要
  - Hybrid 虽然不是最优，但足够好（23.6mm 可接受）

4. Line 的问题

【为什么 Hybrid 是最优解？】
================================================================================

从这张图可以看出：

1. Hybrid 实现了最优的性能平衡
   - 不追求单一指标的极致
   - 追求综合性能的最优
   - 特别是运动平滑性（最重要）

2. Hybrid 的控制策略最合理
   - Bump 负责速度：平滑的速度控制
   - Line 负责转向：精确的方向控制
   - 两者独立运行，不互相干扰
   - 达到最优的速度-转向平衡

3. Hybrid 适合实际应用
   - 运动平滑 → 能耗低、磨损小
   - 可重复性好 → 行为可预测
   - 没有短板 → 适应各种场景
   - 这是实际系统最需要的特性

4. Hybrid 弥补了单一传感器的缺陷
   - Line 在直线上差 → Hybrid 在直线上好（假设 A）
   - Bump 在曲线上差 → Hybrid 在曲线上好（假设 C）
   - Hybrid 是真正的全能选手
