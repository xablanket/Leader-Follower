1. SENSOR PRINCIPLE COMPARISON
================================================================================

[Line Sensor (IR Infrared Sensor)]
  Working Principle:
    - Judges distance by IR light intensity from Leader
    - Direction determined by intensity difference between left/right sensors
  
  Technical Features:
    - Analog output (0-1023)
    - Detection range: Intensity 30-120
    - Fast response
  
  Main Defects:
    ✗ Insensitive to distance changes (intensity ∝ 1/distance², non-linear)
    ✗ Susceptible to ambient light interference
    ✗ Direction judgment relies on tiny signal differences, low precision
    ✗ High signal fluctuation, difficult to provide stable control

[Bump Sensor (Capacitive Proximity Sensor)]
  Working Principle:
    - Measures distance via capacitor charge/discharge time
    - Discharge time proportional to distance, high precision
  
  Technical Features:
    - Digital measurement (time in microseconds)
    - Detection range: 400-1700 μs
    - Independent left/right measurement, reliable direction info
  
  Main Advantages:
    ✓ Sensitive to distance changes (linear relationship)
    ✓ Strong anti-interference capability
    ✓ Direction judgment based on independent measurements, high precision
    ✓ Stable signal, enables precise control

================================================================================
3. WHY IS BUMP SENSOR SUPERIOR?
================================================================================

[Physical Principle Advantages]
  1. Measurement principle better suited for distance measurement
     • Bump: Discharge time ∝ Distance (linear relationship)
     • Line: Light intensity ∝ 1/Distance² (non-linear, low sensitivity at close range)

  2. Strong anti-interference capability
     • Bump: Capacitive measurement, unaffected by ambient light
     • Line: IR light susceptible to sunlight and reflective objects

  3. More reliable direction judgment
     • Bump: Independent left/right measurement, large difference (hundreds of μs)
     • Line: Relies on tiny light intensity difference (may be only a few ADC units)

[Control System Advantages]
  1. More stable distance control PID
     • Bump sensor linear characteristics make PID parameter tuning easier
     • Stable signal reduces control output jitter

  2. More precise steering control
     • Large left/right sensor difference allows reasonable steering gain setting
     • No incorrect steering due to signal reversal

  3. Better robustness
     • Stable sensor readings, control system less likely to diverge
     • Even with noise, minimal impact on system

================================================================================
4. WHY DOES LINE SENSOR PERFORM POORLY?
================================================================================

[Root Cause Analysis]
  1. Inherent limitations of optical sensors
     • IR light intensity decays with square of distance
     • Insufficient sensitivity in following distance range (30-50cm)
     • Small signal variation range (30-120), limited resolution

  2. Environmental sensitivity
     • Different floor reflectivity affects readings
     • Ambient light (sunlight, lamps) causes interference
     • Leader robot attitude changes affect IR emission direction

  3. Direction judgment difficulties
     • Too small difference between left/right sensor signals
     • Signal noise may exceed effective signal
     • Leads to unstable steering commands, robot oscillates left/right

[Experimental Data Evidence]
  • Lateral deviation: Line Sensor Y std dev is 3.6× Bump Sensor
  • Heading stability: Line Sensor max heading deviation is 5.7× Bump Sensor
  • Sensor fluctuation: Line Sensor CV is 0.8× Bump Sensor

  These data clearly demonstrate Line Sensor is unsuitable for straight-line following tasks.
