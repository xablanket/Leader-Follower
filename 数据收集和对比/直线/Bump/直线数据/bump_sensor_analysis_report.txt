【实验概述】
  实验次数: 10 组
  任务: 直线跟随 Leader 机器人
  传感器: Bump Sensor (数字输入，放电时间测量)

【Bump Sensor 主要优势】

  ✓ 优势1: 横向偏离小
     - 平均Y偏离标准差: 3.19 mm
     - 平均最大Y偏离: 10.20 mm
     - 评价: 优秀

  ✓ 优势2: 航向角稳定
     - 平均最大角度偏差: 4.38°
     - 评价: 优秀

  ✓ 优势3: 传感器读数稳定
     - Bump传感器变异系数: 18.36%
     - 评价: 良好

  ✓ 优势4: 速度控制平稳
     - 平均速度变化: 78.43 counts/s
     - 停顿比例: 0.0%

【结论】
  Bump Sensor 在直线跟随任务中表现优异:
  1. 对距离变化敏感，横向偏离控制精确
  2. 方向控制稳定，航向角波动小
  3. 传感器读数稳定，提供可靠的控制信号
  4. 速度控制平滑，运动效率高

  ★ Bump Sensor 明显优于 Line Sensor 用于直线跟随任务
